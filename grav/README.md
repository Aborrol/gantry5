== Grav playground theme

How to install it:

- Enter to Gantry Playground ./grav directory and run in command line:

```
ln -s ../css
ln -s ../css-compiled
ln -s ../fonts
ln -s ../images
ln -s ../js
ln -s ../scss
ln -s ../wordpress/nucleus
ln -s ../wordpress/test
```

- Symbolically link the theme to your site:

```
cd {PATH_TO_GRAV}
ln -s {PATH_TO_HERE} user/themes/nucleus
```
